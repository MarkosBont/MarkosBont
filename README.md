### Selected work

**[gradcam-guided-mammography](https://github.com/MarkosBont/gradcam-guided-mammography)**
My dissertation. A ResNet-50 is fed its own Grad-CAM saliency map as a second
input channel, recursively, on the theory that showing a model where it looked
last step helps guide it on the current step. The ablation proves otherwise: a control
that takes the same optimiser steps with a zeroed channel performs just as well.

**[SpeechToText](https://github.com/MarkosBont/SpeechToText)**
Greek medical exam transcription, in daily use by a clinical team. Whisper for
transcription, GPT for polishing, Supabase for auth and per-user data
storing. 

**[Summary_RAG_Engine](https://github.com/MarkosBont/Summary_RAG_Engine)**
A retrieval-augmented pipeline that turns SkillCorner tracking data into
scouting-ready player summaries. In production with 200+ professional football
clubs.

**[skillcornerviz](https://pypi.org/project/skillcornerviz/)**
Open-source visualisation library for football tracking data, built during my
second SkillCorner internship. 10k+ installs on PyPI. `pip install skillcornerviz`
